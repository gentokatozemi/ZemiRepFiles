## 軽部将伍　「オンライン投票の投票参加促進効果と政府・技術への信頼」 明治大学政治経済学部加藤言人ゼミナール2025年度卒業論文.

### 分析再現ファイル一覧

1. **分析再現コード_軽部将伍.R**：R言語による分析コードファイル。記述統計・推測統計・回帰分析等を実行する。

2. **experiment_origin_subset.sav**：筆者によるサーベイ実験の元データ（実験に関連する変数のみを抽出、実験実施日：2026年1月4日～5日）。

3. **major_results_2020.xlsx**：このファイルは、総務省統計局が整備する、「[政府統計の総合窓口e‐Stat](https://www.e-stat.go.jp/regional-statistics/ssdsview/municipality)」で公開されている、2020年国勢調査の「国勢調査 / 都道府県・市区町村別の主な結果 / 都道府県・市区町村別の主な結果」のxlsxファイルである。分析にあたり人口データとして、交絡変数の「人口数」「人口密度」「15歳未満の人口比率」に用いた。
