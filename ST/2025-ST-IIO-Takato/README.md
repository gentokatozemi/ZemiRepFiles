## 飯尾天翔　「地方議会選挙における再立候補行動の決定要因」 明治大学政治経済学部加藤言人ゼミナール2025年度卒業論文.

### 分析再現ファイル一覧

1. **分析再現コード_飯尾天翔.R**：R言語による分析コードファイル。 Rの分析コードでは、選挙データの前処理や変数作成、回帰分析とその結果の可視化を行っている。また、1～4の仮説ごとに設定した変数を実際のデータに基づいて測定し、固定効果を含む回帰分析によって検証を行っている。これによって、候補者の再立候補行動2冠する要因を体系的に分析することができている。

2. **ゼミ_データ.xlsx**： 市議会議員選挙の選挙結果をまとめたExcel上のデータ。このデータは、秋田県内にある10市の市議会議員選挙を対象に選挙結果をまとめたものである。選挙ドットコムなどの選挙情報サイトや各市のHPから筆者が情報を収集した。記録されているデータに限界があるため本研究では、2005年から各市の直近の選挙までを対象期間としている。データの内容としては、自治体、名前、年齢、職業、得票数、現職か新人か、政党所属などをまとめ、回帰分析に用いている。（筆者作成：2026年1月23日最終更新）

3. **ゼミ_職業コーティング.docx**：地縁型社会資本を持つ職業の分類を行うためのコーディングマニュアル。High、Middle、Lowの3つの分類に分けてまとめている。
